### 主要功能
在线考试系统Online Exam，主要使用Vue cli脚手架结合Ant Design Vue搭建前端，使用SpringBoot结合MyBatis搭建后端，并采用MySQL作为数据库，实现登录注册、查看问题、新建问题、查看考试、新建考试、参加考试、考试管理和个人设置等功能，能够基本满足线上测试的需求，并按照功能进行了页面的设计与优化，尽量体现简洁与美观的效果，以实现较好的交互效果。
